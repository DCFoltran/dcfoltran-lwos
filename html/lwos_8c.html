Detailed Description

Main library source for LWOS.

Author:
Al Williams
Date:
30 Oct 2012 Lightweight OS by Al Williams (al.williams@awce.com) A very simple "OS" written in ordinary C that provides cooperative multitasking and is simple enough to port to very small microcontrollers that have a C compiler

The lwos file has its own main() (although you can suppress this by defining LWOS_NOMAIN.
